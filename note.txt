Kamu sedang mengerjakan project GuruPintar (PHP + MySQL + Tailwind CSS).
Fitur "5. Pengaturan Penyimpanan (Google Sheets)" di tab Quiz SUDAH BERFUNGSI.
Sekarang perlu perbaikan UI agar lebih rapi dan user-friendly.

KONTEKS:
- File utama: index.php (bagian renderGSheetSettings)
- API: api/gsheet_settings_get.php, api/gsheet_settings_save.php, api/gsheet_test_connection.php
- Tabel DB: gsheet_settings (id, user_id, mapel, spreadsheet_url, spreadsheet_id, is_active, auto_sync, include_detail)

===========================
PERBAIKAN 1 — Tambah kolom `sheet_title` di tabel gsheet_settings
===========================

ALTER TABLE gsheet_settings ADD COLUMN sheet_title VARCHAR(255) DEFAULT NULL AFTER spreadsheet_id;

Saat guru menyimpan / test koneksi spreadsheet, ambil judul spreadsheet dari Google Sheets API response (endpoint GET https://sheets.googleapis.com/v4/spreadsheets/{spreadsheetId}?fields=properties.title).
Simpan hasilnya ke kolom sheet_title.
Update api/gsheet_test_connection.php → setelah berhasil connect, return juga field "title" dari response.
Update api/gsheet_settings_save.php → terima field sheet_title, simpan ke DB.

===========================
PERBAIKAN 2 — Tabel daftar mapel: ganti URL dengan nama spreadsheet
===========================

Di renderGSheetSettings(), bagian tabel daftar mapel:

SEBELUM (sekarang):
- Kolom: Mata Pelajaran | Google Sheet (tampil URL) | Status | Opsi (checkbox auto sync + detail) | Aksi

SESUDAH:
- Hapus kolom "Opsi" dari tabel (checkbox dipindah ke modal)
- Ganti tampilan tabel dari <table> menjadi card-list (div per row):
  Setiap baris mapel tampilkan:
  - Kiri: icon mapel (kotak warna 34px, rounded-lg, icon tabler di tengah) + nama mapel (font-semibold) + subtitle "X quiz · {sheet_title}" (font kecil abu-abu)
  - Kanan: badge status ("Terhubung" hijau atau "Belum diset" kuning) + tombol icon edit (pencil) + tombol icon buka (external-link, hanya tampil kalau sudah terhubung)

Kalau sheet_title kosong/null, tampilkan "belum ada spreadsheet" dengan warna warning.

Mapping icon per mapel (case-insensitive contains):
- "matematika" → ti-math-function, bg green-100
- "ipa" / "ilmu pengetahuan alam" → ti-flask, bg purple-100  
- "ips" / "ilmu pengetahuan sosial" → ti-world, bg blue-100
- "bahasa indonesia" → ti-book, bg coral/orange-100
- "bahasa inggris" / "english" → ti-language, bg teal-100
- "bahasa jawa" / "bahasa daerah" → ti-language, bg teal-100
- default → ti-notebook, bg gray-100

PERBAIKAN 4 — Modal tambah/edit: perbaiki layout
===========================

Modal "Tambah Mata Pelajaran" yang sekarang layoutnya 2 kolom horizontal (mata pelajaran + URL sejajar) → ubah jadi VERTIKAL (stack):

Urutan field dari atas ke bawah:
1. Header: "Tambah mata pelajaran" + tombol close (X) di kanan

6. Checkbox row (horizontal): ☑ Auto sync | ☐ Sertakan detail jawaban
7. Info box kecil (bg gray-100, rounded, icon ti-bulb): "Opsi 'auto sync' akan mengirim data ke sheet setiap kali siswa submit jawaban. 'Sertakan detail jawaban' menambah kolom J1, J2, J3... di sheet kelas."
