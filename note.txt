## C. AUTO-DERIVE FASE DARI JENJANG + KELAS

Field "Fase" dihapus dari UI tapi state.paket.fase harus tetap terisi otomatis.

Tambahkan fungsi helper:

```javascript
function deriveFase(jenjang, kelas) {
  // Fase Kurikulum Merdeka
  if (jenjang === 'PAUD') return 'Fondasi';
  if (jenjang === 'TK') return 'Fondasi';
  
  if (jenjang === 'SD/MI') {
    if (['Kelas 1', 'Kelas 2'].includes(kelas)) return 'Fase A';
    if (['Kelas 3', 'Kelas 4'].includes(kelas)) return 'Fase B';
    if (['Kelas 5', 'Kelas 6'].includes(kelas)) return 'Fase C';
  }
  
  if (jenjang === 'SMP/MTs') {
    return 'Fase D';
  }
  
  if (jenjang === 'SMA/MA') {
    if (kelas === 'Kelas 10') return 'Fase E';
    if (['Kelas 11', 'Kelas 12'].includes(kelas)) return 'Fase F';
  }
  
  if (jenjang === 'SMK/MAK') {
    if (kelas === 'Kelas 10') return 'Fase E';
    if (['Kelas 11', 'Kelas 12', 'Kelas 13'].includes(kelas)) return 'Fase F';
  }
  
  return '';
}
```

Lalu di handler onChange Jenjang & Kelas, panggil:

```javascript
state.paket.fase = deriveFase(state.paket.jenjang, state.paket.kelas);
saveDebounced(true);
render();
```

PENTING: Pastikan logic lain yang pakai `state.paket.fase` (mis. prompt OpenAI, kop dokumen Word/PDF) tetap berfungsi dengan fase yang auto-derived.

## D. TAMBAHKAN BADGE DI PENGATURAN LANJUTAN

Saat user sudah mengisi setting di Pengaturan Lanjutan, tampilkan indicator visual supaya jelas:

```html
<span class="...badge..." id="lanjutanBadge">
  ${countFilledLanjutan()} pengaturan
</span>
```

Di mana `countFilledLanjutan()`:

```javascript
function countFilledLanjutan() {
  const p = state.paket || {};
  let count = 0;
  if ((p.namaGuru || '').trim()) count++;
  if ((p.namaSekolah || '').trim()) count++;
  if (p.logo) count++;
  if ((p.tahunAjaran || '').trim()) count++;
  if ((p.judul || '').trim()) count++;
  if ((p.semester || '').trim()) count++;
  if ((p.perintahKhusus || '').trim()) count++;
  return count > 0 ? `${count} terisi` : '7 pengaturan';
}
```

Saat ada yang terisi, badge berubah jadi "3 terisi" dengan warna primary (biru) untuk visual feedback.

## E. PRE-FILL DARI PROFIL USER (Phase 5 sebelumnya)

Saat first time buka halaman, pre-fill dari profil:

```javascript
function initIdentitasFromProfile() {
  if (!state.paket.namaGuru && __userProfile.nama) {
    state.paket.namaGuru = __userProfile.nama;
  }
  if (!state.paket.namaSekolah && __userProfile.nama_sekolah) {
    state.paket.namaSekolah = __userProfile.nama_sekolah;
  }
  if (!state.paket.jenjang && __userProfile.jenjang) {
    state.paket.jenjang = __userProfile.jenjang;
  }
  if (!state.paket.tahunAjaran) {
    const now = new Date();
    const y = now.getFullYear();
    const m = now.getMonth() + 1;
    state.paket.tahunAjaran = m >= 7 ? `${y}/${y+1}` : `${y-1}/${y}`;
  }
}
```

Panggil di state init.
